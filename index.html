AdventureEdit

Level Editor für Kinect Adventures (Xbox 360)

Alpha Version

Windows

Executable

Xbox 360

Level Editing

Python

Backend

Features

  • Öffnen von Kinect Adventures Level-Dateien.
  • Anzeige aller gefundenen Objekte.
  • Objekte werden nach Gruppen kategorisiert:
    • Track
    • Obstacle
    • Timer / Logic
  • Anzeige des internen Namens jedes Objekts.
  • Bearbeiten von Objektkoordinaten.
  • Speichern der Änderungen zurück in die Level-Datei.

Schnellstart

  1. Die Kinect Adventures ISO extrahieren.
  2. Die gewünschte Level-Datei entpacken.
  3. AdventureEdit.exe starten.
  4. Die Level-Datei öffnen.
  5. Objekte bearbeiten und speichern.
